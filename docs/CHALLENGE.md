# Desafio - Vaga Cientista de Dados - Jul/2026 - Carpa Analytics

O conjunto de dados desse desafio é referente ao conjunto de empréstimos intermediado pela empresa LendingClub, em determinado período de tempo. 

A coluna `is_bad` define se um empréstimo foi completamente pago ou não. Valor "1" significa um mal pagador, isto é, inadimplente. O valor "0", bom pagador. A demais, são variáveis explicativas e um dicionário com as descrição dessas variáveis pode ser encontrado no arquivo [`descritivo_dados.md`](./descritivo_dados.md).

Usando Python em um Jupyter Notebook, enderece as perguntas. Ao final, envie os artefatos gerados nessas análise (notebook, e gráficos gerados em um único pdf). Caso a pergunta solicite um artefato gráfico e você queira utilizar esse recurso, inclua-o no anexo, uma vez que não será possível anexar na própria pergunta, no formulário. As perguntas são numeradas com o objetivo de facilitar o envio e referência aos artefatos.

O objetivo desse desafio não é comparar a performance do modelo entre os candidatos e sim entender a forma de raciocinar e trabalhar de cada um.
